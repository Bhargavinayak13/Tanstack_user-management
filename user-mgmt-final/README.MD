# 🚀 TanStack User Management System

A modern and responsive User Management System built with React, TypeScript, Vite, and TanStack Router. This application provides a clean interface to manage users with CRUD operations, routing, and state management.

---

## 📌 Features

- 🔐 Authentication (Login Page)
- 👥 View All Users
- ➕ Add New Users
- ✏️ Edit Existing Users
- 🗑️ Delete Users
- 🔍 Search & Filter Users
- 📱 Responsive Design
- ⚡ Fast Performance with Vite
- 🛣️ Type-safe Routing using TanStack Router
- 📦 API Integration
- 🎨 Clean UI

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/tanstack-user-management.git
```

Go to project folder

```bash
cd tanstack-user-management
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

Open

```
http://localhost:5173
```

---

## 🏗️ Build

```
npm run build
```

Preview production build

```
npm run preview
```
